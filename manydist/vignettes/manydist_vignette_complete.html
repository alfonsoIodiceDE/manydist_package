1 Introduction

The manydist package provides a flexible framework for distance-based learning with continuous, categorical, and mixed-type predictors. Its core function mdist() computes mixed-variable dissimilarities under a range of presets and custom specifications, while the MDist class stores both the resulting distance matrix and the associated metadata. The package also integrates with the tidymodels ecosystem through a recipe step, step_mdist(), and a custom distance-based nearest-neighbour model, nearest_neighbor_dist().

In this vignette we use the palmerpenguins data to show:

  1. How to compute mixed-type distances with mdist() in base R.
  2. How to use step_mdist() to obtain a distance-based representation in a recipe.
  3. How to fit and tune a k-nearest neighbours model based on manydist via nearest_neighbor_dist() and tune_grid().
  4. How the same distance machinery can be used consistently across direct and tidymodels-based workflows.

The focus is on classification of penguin species, but the same ideas apply to other supervised or unsupervised tasks that rely on pairwise distances.

3 Variable influence diagnostics with lovo_mdist()

4 Comparing LOVO diagnostics across methods

The function compare_lovo_mdist() extends the leave-one-variable-out diagnostic to multiple distance specifications. This is useful when we want to assess how variable contributions change across alternative mixed-type distances.

Here we compare two presets:

  • "gower", a standard mixed-type distance,
  • "unbiased_dependent", an association-aware distance that enforces commensurability.

The output is an MDistLOVOCompare object that supports print(), summary(), and autoplot().

lovo_cmp <- compare_lovo_mdist(
  x = penguins_clean |> select(-species),
  methods = list(
    gower = list(
      preset = "gower"
    ),
    u_dep = list(
      preset = "unbiased_dependent"
    ),
    custom = list(
      distance_cont = "manhattan",
      distance_cat  = "HL",
      commensurable = FALSE
    )
  )
)

5 Base R: using mdist() directly

x_train <- peng_train |>
  select(-species)
x_test  <- peng_test |>
  select(-species)
y_train <- peng_train$species
y_test  <- peng_test$species
d_obj <- mdist( x = x_train, new_data = x_test, preset = "unbiased_dependent" )

9 Summary

In this vignette we showed how manydist can be used for distance-based learning on mixed data using mdist(), step_mdist(), and nearest_neighbor_dist() within the tidymodels ecosystem.
